# 📚 FIAP Logic Foundations / Nivelamento em Lógica de Programação

This repository is dedicated to storing and documenting all the foundational exercises from the Analysis and Systems Development (ADS) course at FIAP. It brings together dozens of practical challenges solved in Python and Java, focusing on the core principles of programming logic.

## 🇺🇸 English Version

### About the Project
This repository documents my foundational journey into programming logic as part of the Systems Analysis and Development (ADS) curriculum at FIAP. It serves as a comprehensive logbook of exercises, algorithms, and core programming concepts implemented in Python and Java. The goal is to demonstrate consistency, evolution, and mastery of fundamental logic structures.

### Key Learnings and Content
Throughout these modules, I have mastered essential algorithmic thinking and its practical application:

*   **Algorithmic Thinking & Flowcharts:** Understanding the history of programming, abstract algorithms, pseudocode, and visual representation of logic using flowcharts.
*   **Data I/O & Variables:** Implementation of basic input/output commands (`print`, `input`, `Scanner`), variable types (`int`, `float`, `bool`, `str`), arithmetic operators, and type casting.
*   **Conditional Structures:** Deep understanding of decision-making processes using simple `if`, compound `if/else`, chained `if/elif/else`, and Java's `switch-case` statements to control program flow based on conditions.
*   **Loops (Repetition Structures):** Mastery of indefinite (`while`, `do-while`, `while True`) and definite (`for`) loops. Projects include summing numbers until zero is entered, electoral voting systems, factorials, and generating multiplication tables.

## 🇧🇷 Versão em Português

### Sobre o Projeto
Repositório dedicado ao armazenamento e documentação de todos os exercícios de nivelamento do curso de Análise e Desenvolvimento de Sistemas (ADS) da FIAP. Este projeto reúne dezenas de desafios práticos resolvidos em Python e Java, focando nos fundamentos da lógica de programação. O objetivo é demonstrar minha evolução técnica, consistência nos estudos e a aplicação de boas práticas de código (clean code) aprendidas nas aulas.

### Aprendizados Chave e Conteúdo
Ao longo dos módulos, dominei o pensamento algorítmico essencial e sua aplicação prática:

*   **Pensamento Algorítmico e Fluxogramas:** Compreensão da história da programação, algoritmos abstratos, pseudocódigo e representação visual de lógica.
*   **Entrada/Saída e Variáveis:** Implementação de comandos básicos de I/O (`print`, `input`, `Scanner`), tipos de dados (`int`, `float`, `bool`, `str`), operadores aritméticos e conversão de tipos (casting).
*   **Estruturas Condicionais:** Entendimento profundo de processos de tomada de decisão usando `if` simples, `if/else` composto, `if/elif/else` encadeado e `switch-case` (Java) para controlar o fluxo do programa.
*   **Laços de Repetição (Loops):** Domínio de loops indefinidos (`enquanto-faça`, `faça-enquanto`, `while True`) e definidos (`para`/`for`). Os projetos incluem somar números até digitar zero, sistemas de votação eleitoral, cálculo de fatorial e tabuadas.
