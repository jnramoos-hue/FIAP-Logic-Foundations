# Python Logic Fundamentals
## Overview

This directory is part of my software engineering portfolio and contains Python logic exercises originally developed during my academic studies.

The goal of this work is to transform classroom-style exercises into a cleaner, more professional, and internationally readable portfolio structure, with a focus on:

- English naming conventions
- organized folder structure
- improved readability
- better presentation for recruiters and tech leads

## What Was Done Today

Today, I started the migration of my Python exercises from a flat academic structure into a categorized portfolio structure.

- removed the old legacy Python directory structure
- created a new portfolio-focused directory called `python-logic-fundamentals`
